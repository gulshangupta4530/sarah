C PROGRAMS

Basic Input-Output Questions

1. Write a program to print the message: Today I made my first C Program.

Code:

#include <stdio.h>
void main() {
  printf("Today I made my first C program.");
  getch();
}

Output:

Today I made my first C program.

2. Write a program to read a character from the keyboard and display it on the screen.

Code:

#include <stdio.h>
#include <conio.h>
void main() {
  char ch;
  clrscr();
  printf("Type a character: ");
  scanf("%c", &ch);
  printf("The character is: %c", ch);
  getch();
}

Output:

Type a character: k
The character is: k

3. Write a program to print your name at the center of the first line.

Code:

#include <stdio.h>
void main() {
  printf("\t\t\tSarah\n");
  getch();
}

Output:

                        Sarah

4. Write a program to store a single quote and print that variable to show single quote on output screen.

Code:

#include <stdio.h>
#include <conio.h>
void main() {
  char ch = '\'';
  clrscr();
  printf("Single quote is: %c", ch);
  getch();
}

Output:

Single quote is: '

c
d

6. Write a program for printing your name on output screen but your name should blink 5 times

Code:

#include <stdio.h>
#include <conio.h>
#include <dos.h>
void main(){
    int i;
    for(i=1;i<=5;i++){
     
        printf("abc");
        delay(500);
       clrscr();

       delay(500);
    }
}

Output:

  abc

Code:
